## How to use 

1. Clone or download repo.
2. Go to the folder. Make sure that you unzipped it in case if you choosed "download zip" insted of clone.

```bash
cd ./E-cart-FatAPI-Next.js
```
3. Run 

```bash
docker-compose --build up
```
